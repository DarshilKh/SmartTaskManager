# Smart Task Manager

A scalable Java-based task management system built using **Spring, JSP, and Servlets**, designed to efficiently manage and prioritize tasks with a clean layered architecture.

---

## 🚀 Features

- Create, update, delete tasks (CRUD operations)
- Set deadlines and track task timelines
- Priority-based task organization
- Clean layered architecture (Controller → Service → DAO)
- Maintainable and scalable backend design

---

## 🛠 Tech Stack

- **Backend:** Java, Spring Framework, Servlets
- **Frontend:** JSP
- **Architecture:** MVC (Model-View-Controller)
- **Build Tool:** Maven (if used)
- **Version Control:** Git & GitHub

---
## ⚙️ Installation & Setup

- 1. Clone the repository: git clone https://github.com/your-username/SmartTaskManager.git
- 2.Open in IDE (IntelliJ / Eclipse)
- 3.Configure server: Apache Tomcat (recommended)
- 4.Build & run the project: Deploy on Tomcat server
- 5.Access in browser: http://localhost:8080/SmartTaskManager
